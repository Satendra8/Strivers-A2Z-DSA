Stack is a data structure that follows the Last In First Out (LIFO) rule.

Stack Functions:
    i. push
    ii. pop
    iii. top
    iv. size



Queue is a data structure that follows the First In First Out (FIFO) rule.

Queue Functions:
    i. push
    ii. pop
    iii. rear
    iv. front
    v. size


Pattern 1: take array size fixed
    1. implement stack using array
    2. implement queue using array (make circular using %)

Pattern2: making dynamic using linked list
    1. implement stack using linkedlist (mark top as head node always)
    2. implement queue using linkedlist (make circular using %)
